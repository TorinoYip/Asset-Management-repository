# Asset Intellgent 网页

这是 Asset Intellgent 的静态网页，用于呈现资产总览、经营结果和运营表现，并验证页面层级与跨页导航。

## 打开方式

直接打开 `index.html` 即可；也可以从仓库根目录启动任意静态文件服务：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000/apps/web/`。

## 已实现

- `#overview`：资产总览；
- `#business`：经营看板；
- `#operations`：运营看板；
- 左侧导航与返回入口；
- 经营摘要、运营摘要下钻；
- 截止月份、省份、电站和资产类型筛选上下文；
- 风电/储能同级 Tab；
- 经营指标、趋势、排名联动；
- 风电单机与储能单站的运营指标比较；
- 地图点位选择和异常资产定位。

## 文件边界

- `index.html`：语义结构和页面内容；
- `styles.css`：视觉层级与响应式布局；
- `app.js`：假数据、hash 路由和交互状态。

指标名称与层级参考信息架构及指标字典。正式开发时应以数据服务替换 `app.js` 中的数据对象，并将业务计算迁移到领域层。
