# Asset Intellgent 网页

这是 Asset Intellgent 的静态网页，用于验证资产管理看板的五页信息架构、页面层级与跨页导航。

## 打开方式

直接打开 `index.html` 即可；也可以从仓库根目录启动任意静态文件服务：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000/apps/web/`。

## 已实现

- `#overview`：资产看板，采用“2 个一级主区 + 3 个二级概况”的结构；
- `#revenue`：营收总览；
- `#types`：按类型分析；
- `#stations`：按站点分析；
- `#mengxi`：蒙西日报入口，本轮仅保留页面骨架；
- 左侧导航与返回入口；
- 经营摘要向营收总览下钻；
- 截止月份、省份、电站和资产类型筛选上下文；
- 四项全局筛选会同步更新资产看板与营收总览中的卡片、地图、趋势和表格；
- 河北、内蒙古、山东、浙江四个省份的地图点位及悬浮电站摘要；
- 资产构成表与可筛选、可多选的资产明细横表；
- 月份下拉对比表；
- YTD / 具体月份的同类资产横向比较；
- 风电单机与储能单站的同级运营指标比较；
- 站点经营、运营、运维摘要与结算单拆解。

## 文件边界

- `index.html`：语义结构和页面内容；
- `styles.css`：视觉层级与响应式布局；
- `app.js`：示例数据、hash 路由和交互状态。

指标名称与层级参考信息架构及指标字典。正式开发时应以数据服务替换 `app.js` 中的数据对象，并将业务计算迁移到领域层。
