# 指标字典（V2）

## 1. 资产对象统一定义

| 资产类型 | `asset_level` | 单体分析对象 | 最小时间粒度 |
|---|---|---|---|
| 风电 | asset | 单台风机 | 月度 |
| 储能 | asset | 单个储能电站（单站） | 月度 |

单机和单站必须共享以下公共字段：

- `asset_id`
- `asset_name`
- `asset_type`
- `region`
- `project_id`
- `year`
- `month`
- `status`
- `actual_value`
- `target_value`
- `data_updated_at`

风电场/项目是单机的聚合维度，不改变单机与储能单站在功能层级上的同级关系。

## 2. 公共时间口径

| 字段/指标 | 定义 | 备注 |
|---|---|---|
| 统计月份 | 当前页面所选月份 | 页面显示“截至 YYYY 年 MM 月” |
| 月度实际值 | 单个自然月的实际值 | 最小分析粒度 |
| 实际 YTD | 年初至所选月份的实际累计值 | `actual_ytd(M)=Σ monthly_actual(1..M)` |
| 累计目标 | 年初至所选月份的计划累计值 | 判断当前进度 |
| 全年目标 | 完整年度目标值 | 判断全年完成进度 |
| YTD差距 | 实际YTD − 累计目标 | 成本类指标反向解释 |
| 累计目标完成率 | 实际YTD ÷ 累计目标 | 收入/毛利越高越好；成本单独标识方向 |
| 全年目标完成进度 | 实际YTD ÷ 全年目标 | 不等同于累计目标完成率 |
| 月度目标 | 当月累计目标 − 上月累计目标 | 累计目标按月保存时推导 |

## 3. 资产总览指标

| 指标 | 单位 | 粒度 |
|---|---:|---|
| 风机数量 | 台 | 组合 / 区域 / 项目 |
| 储能单站数量 | 个 | 组合 / 区域 |
| 风电装机容量 | MW | 组合 / 区域 / 项目 / 单机 |
| 储能额定功率 | MW | 组合 / 区域 / 单站 |
| 储能额定容量 | MWh | 组合 / 区域 / 单站 |
| YTD风电电量 | MWh | 组合 / 区域 / 项目 / 单机 |
| YTD充电量 / 放电量 | MWh | 组合 / 区域 / 单站 |
| 资产状态 | 源数据状态枚举 | 单机 / 单站 |
| 纬度 / 经度 | 十进制度 | 风电项目 / 储能单站；用于地图 |

## 4. 经营指标

| 指标 | 建议定义 | 单位 | 方向 |
|---|---|---:|---|
| 总收入 | 已有收入分项之和 | 元 | 越高越好 |
| 总成本 | 已有成本分项之和 | 元 | 越低越好 |
| 总毛利 | 总收入 − 总成本 | 元 | 越高越好 |
| 收入差距 | 实际收入YTD − 累计收入目标 | 元 | 正向 |
| 成本差距 | 实际成本YTD − 累计成本目标 | 元 | 负向；超支为异常 |
| 毛利差距 | 实际毛利YTD − 累计毛利目标 | 元 | 正向 |
| 经营状态 | 依据完成率和方向性规则生成 | 状态 | 阈值后续确认 |

风电和储能采用同样的目标、实际、差距、趋势、排名和状态功能，只保留不同的收入成本分项。

## 5. 风电单机运营指标

| 指标 | 建议定义 | 单位 |
|---|---|---:|
| 发电量/上网电量 | 按现有计量口径使用 | MWh |
| 风机可利用率 | 可运行时间 ÷ 统计时间 | % |
| 限电量 | 因调度限电减少的可发电量 | MWh |
| 限电率 | 限电量 ÷ 理论可发电量 | % |
| 计划损失率 | 计划停机损失电量 ÷ 理论可发电量 | % |
| 非计划损失率 | 非计划停机损失电量 ÷ 理论可发电量 | % |
| 风机故障 MTTR | 故障修复总时长 ÷ 已修复故障数 | h |

## 6. 储能单站运营指标

| 指标 | 建议定义 | 单位 |
|---|---|---:|
| 充电电量 | 统计月内储能单站充电侧电量 | MWh |
| 放电电量 | 统计月内储能单站放电侧电量 | MWh |
| 日均充放次数 | 等效完整循环次数 ÷ 统计天数 | 次/日 |
| 充放电效率 | 放电量 ÷ 充电量 | % |
| 容量可利用率 | 可用容量 ÷ 额定容量 | % |
| 时间可利用率 | 可用时间 ÷ 统计时间 | % |
| 储能可利用率 | 源系统已有综合可利用率 | % |
| 储能故障 MTTR | 故障修复总时长 ÷ 已修复故障数 | h |
| 非计划损失时长 | 非计划不可用时长之和 | h |

单机与单站的专业指标不同，但展示组件、排序、状态、月度比较和详情能力一致。

## 7. 公共维度

- 组织：集团、区域/项目公司。
- 资产：资产类型、风电项目、风机单机、储能单站。
- 时间：年份、截至月份。
- 目标版本：全年目标、累计目标及其生效版本。
- 状态：资产状态、经营状态、运营异常状态。
- 地理：风电项目或储能单站经纬度。
