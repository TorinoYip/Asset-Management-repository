# 指标字典（V2）

## 1. 公共时间口径

| 字段/指标 | 定义 | 备注 |
|---|---|---|
| 统计月份 | 当前页面所选月份 | 页面显示“截至 YYYY 年 MM 月” |
| 月度实际值 | 单个自然月的实际值 | 最小分析粒度 |
| 实际 YTD | 年初至所选月份的实际累计值 | `actual_ytd(M)=Σ monthly_actual(1..M)` |
| 累计目标 | 年初至所选月份的计划累计值 | 用于判断当前进度是否达标 |
| 全年目标 | 完整年度目标值 | 用于判断全年完成进度 |
| YTD差距 | 实际YTD − 累计目标 | 成本类指标需反向解释 |
| 累计目标完成率 | 实际YTD ÷ 累计目标 | 收入/毛利越高越好；成本需单独标识方向 |
| 全年目标完成进度 | 实际YTD ÷ 全年目标 | 不等同于累计目标完成率 |
| 月度目标 | 当月累计目标 − 上月累计目标 | 累计目标按月保存时可推导 |

## 2. 资产总览指标

| 指标 | 单位 | 粒度 |
|---|---:|---|
| 风电站点数 | 个 | 组合 / 区域 |
| 储能电站数 | 个 | 组合 / 区域 |
| 风电装机容量 | MW | 组合 / 区域 / 站点 |
| 储能额定功率 | MW | 组合 / 区域 / 站点 |
| 储能额定容量 | MWh | 组合 / 区域 / 站点 |
| YTD上网电量 | MWh | 组合 / 区域 / 风电站点 |
| YTD充电量 / 放电量 | MWh | 组合 / 区域 / 储能电站 |
| 资产状态 | 正常/关注/异常等 | 站点；枚举需与源数据一致 |
| 纬度 / 经度 | 十进制度 | 站点；用于资产地图 |

## 3. 经营指标

| 指标 | 建议定义 | 单位 | 方向 |
|---|---|---:|---|
| 总收入 | 已有各收入分项之和 | 元 | 越高越好 |
| 总成本 | 已有各成本分项之和 | 元 | 越低越好 |
| 总毛利 | 总收入 − 总成本 | 元 | 越高越好 |
| 收入差距 | 实际收入YTD − 累计收入目标 | 元 | 正向 |
| 成本差距 | 实际成本YTD − 累计成本目标 | 元 | 负向；超支为异常 |
| 毛利差距 | 实际毛利YTD − 累计毛利目标 | 元 | 正向 |
| 站点经营状态 | 依据完成率和方向性规则生成 | 状态 | 阈值后续确认 |

风电和储能使用同一经营语义，但各自保留已有收入、成本分项。V2 不在界面层重新定义会计口径。

## 4. 风电运营指标

| 指标 | 建议定义 | 单位 | 粒度 |
|---|---|---:|---|
| 上网电量 | 结算口径并网点上网电量 | MWh | 月 / 站点；单机口径需区分发电量 |
| 风机可利用率 | 可运行时间 ÷ 统计时间 | % | 月 / 站点 / 单机 |
| 限电量 | 因调度限电减少的可发电量 | MWh | 月 / 站点 / 单机 |
| 限电率 | 限电量 ÷ 理论可发电量 | % | 月 / 站点 / 单机 |
| 计划损失率 | 计划停机损失电量 ÷ 理论可发电量 | % | 月 / 站点 / 单机 |
| 非计划损失率 | 非计划停机损失电量 ÷ 理论可发电量 | % | 月 / 站点 / 单机 |
| 风机故障 MTTR | 故障修复总时长 ÷ 已修复故障数 | h | 月 / 站点 / 单机 |

## 5. 储能运营指标

储能 V2 最细粒度为“储能电站 × 月份”。

| 指标 | 建议定义 | 单位 |
|---|---|---:|
| 充电电量 | 统计月内储能电站充电侧电量 | MWh |
| 放电电量 | 统计月内储能电站放电侧电量 | MWh |
| 日均充放次数 | 等效完整循环次数 ÷ 统计天数 | 次/日 |
| 充放电效率 | 放电量 ÷ 充电量 | % |
| 容量可利用率 | 可用容量 ÷ 额定容量 | % |
| 时间可利用率 | 可用时间 ÷ 统计时间 | % |
| 储能可利用率 | 源系统已有综合可利用率 | % |
| 储能故障 MTTR | 故障修复总时长 ÷ 已修复故障数 | h |
| 非计划损失时长 | 非计划不可用时长之和 | h |

计量侧、停机剔除规则和综合可利用率公式应继承源系统口径，并在数据接入阶段补充。

## 6. 公共维度

- 组织：集团、区域/项目公司。
- 资产：资产类型、站点、风机单机。
- 时间：年份、截至月份。
- 目标版本：全年目标、累计目标及其生效版本。
- 状态：资产状态、经营状态、运营异常状态。
- 地理：站点经纬度。
